CSS Grid defines a 2-dimensional grid-based layout system Flexbox defines a 1-dimensional system - either vertical or horizontal

Like Flexbox, Grid is good for

  • Dynamic content
  • Responsive layouts
  • Uniform height/width across elements
  • Changing order
